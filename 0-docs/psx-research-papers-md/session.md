# Session Handoff: PSX Research Paper Processing

## Objective
Process papers in psx-research-papers one by one.
For each paper:
1. Rename the PDF to a clean, standardized, lowercase hyphenated filename.
2. Create a matching .md file in psx-research-papers-md with the same base name.
3. Include these mandatory sections in each markdown summary:
- Model Used
- Data Used
- Where the Data Came From
- Limitations
- Research Gap

## Naming Convention
- PDF and markdown base name format:
  lowercase-words-separated-by-hyphens
- Include year suffix when identifiable and useful to disambiguate:
  ...-2022, ...-2025
